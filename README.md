🌱 AgriSmart – AI Powered Smart Agriculture Platform

Project Details

Project Name: AgriSmart

Description:
AgriSmart is an AI-powered smart agriculture platform designed to help farmers make better farming decisions. The system provides crop recommendations, weather insights, and agricultural guidance based on user inputs and data analysis. Its goal is to improve productivity, reduce risks, and promote sustainable farming practices.

Features

- Crop Recommendation
- Weather Information
- Soil-Based Suggestions
- AI-Powered Assistance
- User-Friendly Dashboard

Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
